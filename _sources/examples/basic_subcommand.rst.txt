Basic Subcommand
****************


::

    usage: basic_subcommand.py {foo|bar} [--help]



.. rubric:: Subcommands

.. table::
    :widths: auto

    +---------+-----------+
    | ``foo`` | Print foo |
    +---------+-----------+
    | ``bar`` | Print bar |
    +---------+-----------+


.. rubric:: Optional arguments

.. table::
    :widths: auto

    +--------------------+---------------------------------+
    | ``--help``, ``-h`` | Show this help message and exit |
    +--------------------+---------------------------------+
