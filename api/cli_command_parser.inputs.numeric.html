class cli_command_parser.inputs.numeric.NumRange(type: Optional[Callable[[str], NT]] = None, snap: Union[bool, Any] = False, *, min: Optional[NT] = None, max: Optional[NT] = None, include_min: Union[bool, Any] = True, include_max: Union[bool, Any] = False)[source]ÔÉÅ

Bases: NumericInput[NT]

A range of integers or floats, optionally only bounded on one side.

By default, the min and max behave like the builtin range - the min is inclusive, and the max is exclusive.

Parameters

  • type¬∂ ‚Äì The type for values, or any callable that returns an int/float

  • snap¬∂ ‚Äì If True and a provided value is outside the allowed range, snap to the nearest bound. Respects inclusivity/exclusivity of the bound. Not supported for floats since there is not an obviously correct behavior for handling them in this context.

  • min¬∂ ‚Äì The minimum allowed value, or None to have no lower bound

  • max¬∂ ‚Äì The maximum allowed value, or None to have no upper bound

  • include_min¬∂ ‚Äì Whether the minimum is inclusive (default: True)

  • include_max¬∂ ‚Äì Whether the maximum is inclusive (default: False)
