Simple Flags
************


::

    usage: simple_flags.py [--foo] [--no-bar] [--help]



.. rubric:: Optional arguments

.. table::
    :widths: auto

    +------------------+------------------------------------------------------+
    | ``--foo, -f``    | (default: ``False``)                                 |
    +------------------+------------------------------------------------------+
    | ``--no-bar, -B`` | (default: ``True``)                                  |
    +------------------+------------------------------------------------------+
    | ``--help, -h``   | Show this help message and exit (default: ``False``) |
    +------------------+------------------------------------------------------+
